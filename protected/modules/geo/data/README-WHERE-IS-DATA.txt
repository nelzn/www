Так как дамп базы стран+городов довольно внушительного размера, мы решили вынести его из репы.
Если вы желаете использовать geo модуль - скачайте SQL + файлы данных в эту директорию.
SQL файл содержит базу стран+городов, а файлы - базы geoip. Соотвественно SQL надо импортнуть в базу:

mysql --databas=yupedb -pPASSWORD -uUSER < geo_data.sql
